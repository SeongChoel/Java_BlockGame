Java Block Game (Swing / Java2D)

자바 스윙을 이용한 블록 격파 게임 구현 

이 프로젝트는 객체지향적으로 구성한점
(추상 클래스, 상속 구조, 오버라이딩, 캡슐화, 다형성 등)

화면 전환: 타이틀 → 게임 → 게임오버(+ 다음 스테이지) 모드

충돌·반사: 공–벽/블록/패들 충돌 시 반사 처리

스테이지 난이도: 스테이지가 올라갈수록 공 속도증가, 블록 그리드 확대

스페셜 블록: 일정 확률의 특수 블록 충돌 시 공이 2개 추가 생성

키 입력: 좌/우 방향키로 패들 이동, Space 로 시작/다음 진행
